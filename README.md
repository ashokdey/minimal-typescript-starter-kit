<center>
![TS Logo](_/ts.png)

## Details

- Uses ESLint with TS rules
- Uses Prettier for code formatting
- Uses Concurrently for compilling and running in single go during development


### Setup

- Clone
- `npm i`
- `npm start`

### Notes

- For building, use `npm run start:build`
